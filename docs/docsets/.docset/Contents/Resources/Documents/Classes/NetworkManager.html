NetworkManager

class NetworkManager: NSObject, NetServiceDelegate, NetServiceBrowserDelegate

Class to handle communication with the Arduino. It automatically determines the Arduino’s IP address using Bonjour, sets up a server on the iOS device for receiving measurement data from the Arduino and implements functions to control the Arduino.
