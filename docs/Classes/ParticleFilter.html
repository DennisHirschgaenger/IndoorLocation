  • R

    R is the covariance matrix for the measurement noise. It is a (numAnchors + 2) x (numAnchors + 2) diagonal matrix of the form:

    |D_N 0|
| 0  A|, with
D_N = dist_sig * I_N, I_N being the identity matrix of order N
A = acc_sig * I_2
